# Yêu cầu xây dựng React DataCollection + TanStack Table Framework

Bạn là chuyên gia ReactJS, TypeScript, TanStack Table, React Query.

Hãy xây dựng khung ứng dụng frontend theo kiến trúc sau.

## 1. Mục tiêu

Xây dựng framework quản lý bảng dữ liệu reusable, inspired by Jmix:

- `DataCollection<T>`: quản lý dữ liệu, loading, paging, sorting, filtering, selection
- `AppTable<T>`: adapter binding `DataCollection<T>` vào TanStack Table
- `CompanyTable`, `DepartmentTable`: table nghiệp vụ, tự render UI table
- `CompanyScreen`, `DepartmentScreen`: orchestration màn hình

Không tạo generic `DataTable<T>` dùng trực tiếp ở screen.

3. Nguyên tắc thiết kế
3.1 useDataCollection<T>

Chịu trách nhiệm:

- React Query loading data
- pagination state
- sorting state
- columnFilters state
- globalFilter state

- selectedIds

3.2 useAppTable<T>

Chịu trách nhiệm:

- nhận DataCollection

- bind state vào TanStack Table

5. useDataCollection<T> cần bổ sung cache mutation helpers

Trong:

src/shared/datacollection/useDataCollection.ts

Bổ sung return:

appendItem(item: T): void;
prependItem(item: T): void;
replaceItem(item: T): void;
removeItem(id: string): void;

Yêu cầu:

queryClient.setQueryData<PageResponse<T>>(fullQueryKey, ...)

Behavior:

appendItem:
- append vào cuối items của current page
- total + 1

prependItem:
- thêm vào đầu items của current page
- total + 1

replaceItem:
- tìm theo getRowId
- thay thế item trong current page

removeItem:
- remove theo id
- total - 1

Không invalidate query ngay sau create/update/delete nếu đang muốn giữ UX local current page.
